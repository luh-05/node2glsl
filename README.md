# node2glsl
Utility to convert a shader node-graph to glsl

## Installation
Clone (ssh):
```
git clone git@github.com:luh-05/node2glsl.git
cd node2glsl
git submodule update --init --recurse
```

Clone (https):
```
git clone https://github.com/luh-05/node2glsl.git
cd node2glsl
git submodule update --init --recurse
```
